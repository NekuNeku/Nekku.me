Discord Reimagined

Taking what is already there while slowly but carefully nailing down each element to perfection.

  • Make sure you have BetterDiscord installed.
  • Download the theme via the Download button above.
  • In Discord, go into the User Settings.
  • On the left, click on the "BetterDiscord" tab.
  • On the top, click on the "themes" tab.
  • Click on "Open themes folder".
  • Paste the .css file you downloaded in that folder.
  • Restart BetterDiscord.
  • Go back to the "themes" menu and select "Enable".
